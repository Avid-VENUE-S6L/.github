## What is Avid VENUE S6L?

Avid VENUE S6L is a modular live sound mixing system that delivers the power and flexibility required for the world's most demanding tours, festivals, houses of worship, and broadcast productions . Designed for sound engineers who demand reliability, sound quality, and integration with Pro Tools, the S6L system is built on a distributed architecture using dual-redundant Ethernet AVB networking.

Infrastructure teams managing live production audio benefit from Avid's ecosystem integration, allowing for seamless recording and virtual soundchecks with Pro Tools . System administrators appreciate the modular design with industry-standard I/O cards, the Engine and Stage I/O rack architecture, and the ability to customize the system for any application.

## VENUE S6L System Architecture

The S6L system is built on a distributed architecture where components connect via dual-redundant Ethernet AVB:

| Component | Description |
|-----------|-------------|
| **S6L Control Surface** | Available in 16C, 24C, 24D, 32D, or 48D configurations |
| **E6L Engine** | DSP engine (E6L-112, E6L-144, or E6L-192) |
| **Stage I/O Racks** | Stage 16, Stage 48, or Stage 64 |
| **Pro Tools Workstation** | Dedicated system for recording and playback |
| **I/O Cards** | Analog, AES, MADI, Dante, and Waves TDM |

## Installation Guide

### Prerequisites

- Windows 10/11 for Pro Tools and VENUE software
- AVB Ethernet network (dual-redundant recommended)
- S6L control surface, E6L engine, and Stage I/O racks
- Power conditioning and redundant power supplies

### Step 1: System Assembly

**Step 1:** Position the S6L control surface in the desired location.

**Step 2:** Mount the E6L engine in a rack with proper cooling and power.

**Step 3:** Position Stage I/O racks near stage inputs.

**Step 4:** Connect all components via dual-redundant Ethernet AVB cables.

**Step 5:** Connect all power supplies (dual supplies for engine and rack redundancy).

### Step 2: Network Configuration

**Step 1:** Connect the S6L network using AVB-compliant switches.

**Step 2:** Power up the system in sequence: Stage I/O racks, E6L engine, control surface.

**Step 3:** The system auto-discovers all connected components.

**Step 4:** Confirm all components are visible in the System Configuration screen.

### Step 3: Configure I/O

**Step 1:** Connect audio sources to Stage I/O rack inputs.

**Step 2:** Connect outputs to PA system, monitors, and broadcast feeds.

**Step 3:** Configure input channels and output routing.

**Step 4:** Set up Pro Tools integration for recording and virtual soundchecks.

### Step 4: Software Installation

**Step 1:** Install Pro Tools on the dedicated workstation.

**Step 2:** Install VENUE software on the console.

**Step 3:** Check for firmware updates on the E6L engine and Stage I/O racks.

**Step 4:** Load show files and configure user preferences.
